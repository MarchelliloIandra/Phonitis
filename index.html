About Phonitis

About Phonitis


Phonitis is a voice box project powered by Arduino, created to deliver personal voice messages in a unique way. It was born out of the difficulty of expressing feelings directly and so, Phonitis came. The system is built using an Arduino Uno, DFPlayer Mini, a speaker, push button, and some jumper wires. When the button is pressed, a pre-recorded voice message will play automatically. On this site, you'll find a short usage guide and documentation that shows how this project works. Every part of it was made with care.

How to Use Phonitis

Setup & How to Use

Before using Phonitis, make sure you have these:

  1. The Phonitis Box (fully assembled)
  2. USB cable for the Arduino
  3. Power source (laptop, USB charger, or PowerBank)

Follow these steps to use Phonitis:

  1. Connect the USB cable from the Phonitis box to a power source (laptop, wall charger, or powerbank).
  2. Wait a few seconds for the system to boot up.
  3. Place your ear close to the speaker.
  4. Press the button located on the box.
  5. The voice message will play automatically through the speaker.
